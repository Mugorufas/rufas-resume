## Installation & Setup

1. **Navigate to the project directory:**
   ```bash
   cd c:\Users\HP\Desktop\WEBSITE\resume
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm start
   ```

4. **Open in browser:**
   The app will automatically open at `http://localhost:3000`

## Customization

Edit the following files to update your information:

- **Header Component** ([src/components/Header.js](src/components/Header.js)) - Update name, email, phone, location
- **Education Component** ([src/components/Education.js](src/components/Education.js)) - Update university and graduation date
- **Skills Component** ([src/components/Skills.js](src/components/Skills.js)) - Add/remove technical skills
- **Projects Component** ([src/components/Projects.js](src/components/Projects.js)) - Add your projects
- **Experience Component** ([src/components/Experience.js](src/components/Experience.js)) - Update work experience
- **Certifications Component** ([src/components/Certifications.js](src/components/Certifications.js)) - Add certifications

## Build for Production

```bash
npm run build
```

This creates an optimized production build in the `build` folder.

## Features Included

### Visual Design
- Gradient background (purple to violet)
- Glass morphism effect with backdrop blur
- Smooth animations and transitions
- Hover effects on cards and skill badges

### Responsive Design
- Mobile-first approach
- Tablets and desktop optimized
- Touch-friendly interactions

### Sections
1. **Header** - Name, title, and contact information
2. **Professional Summary** - Brief overview of skills and experience
3. **Education** - Degree and university information
4. **Technical Skills** - Interactive skill badges
5. **Projects** - Showcase of completed projects
6. **Experience** - Work and internship experience
7. **Certifications** - Professional certifications
